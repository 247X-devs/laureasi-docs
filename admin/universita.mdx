## Dove trovarle

Menu laterale → **🏛️ Università**

<Frame caption="Scheda università">
  <img src="/images/admin/universita-scheda.png" alt="Scheda università con dati principali" />
</Frame>

## Tab principali

| Tab | Contenuto |
| --- | --- |
| **Dati principali** | Nome, logo, contatti, descrizione, regione, rating |
| **Data sources** | Sorgenti dati collegate all'ateneo (vedi guida dedicata) |

## Campi utili da conoscere

| Campo | A cosa serve |
| --- | --- |
| Nome Università | Titolo sulla scheda pubblica |
| Slug | URL dell'ateneo (`/universita/[slug]`) |
| Logo | Immagine brand |
| Sito Web / Email / Telefono | Contatti pubblici |
| Descrizione Breve | Testo introduttivo |
| Università Partner | Boost nel quiz di orientamento |
| Moltiplicatore score quiz | Leva avanzata sul ranking (solo admin tecnici) |
| Metadati Aggiuntivi | Coppie chiave-valore extra |

## Esportare in CSV

Dalla lista **Università** apri il menu azioni e scegli **Export**.

1. Seleziona il formato **CSV**
2. Scegli i campi da includere (o lascia il set predefinito)
3. Scarica il file

Puoi esportare tutta la lista o solo le righe selezionate.

## Sedi universitarie

Le sedi fisiche si gestiscono in una collection separata (**Sedi universitarie**) collegata all'ateneo.
