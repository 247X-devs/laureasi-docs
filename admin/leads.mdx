## Dove trovare i Leads

1. Accedi al [pannello admin](https://admin.laureasi.it)
2. Nel menu laterale apri **👥 Utenti & CRM**
3. Seleziona **Leads**

  <img src="/images/admin/sidebar-menu.png" alt="Menu laterale del pannello admin con la sezione Leads" />
</Frame>

## Cosa vedi nella lista

La tabella mostra le informazioni principali di ogni lead:

| Colonna | Significato |
| --- | --- |
| Nome | Nome e cognome del contatto |
| Email | Indirizzo email |
| Telefono | Numero di telefono (se fornito) |
| Fonte | Da dove proviene (sito, form, landing, ecc.) |
| Stato | Fase nel funnel (nuovo, contattato, qualificato…) |
| Data | Quando è arrivato il lead |

## Aprire il dettaglio di un lead

1. Clicca sulla riga del lead che ti interessa
2. Si apre la scheda completa con messaggio, note interne, tracking e metadati

## Aggiornare lo stato

Nella barra laterale della scheda trovi **Stato**. Valori tipici:

| Stato | Quando usarlo |
| --- | --- |
| Nuovo | Lead appena arrivato, non ancora lavorato |
| Contattato | Hai già risposto o chiamato |
| Qualificato | Interesse confermato |
| Convertito | Iscrizione o vendita conclusa |
| Non interessato | Non procede |
| Archiviato | Chiuso senza follow-up |

## Assegnare un lead

Nel campo **Assegnato a** puoi collegare un operatore del team (utente admin) responsabile del follow-up.

## Note interne

Usa il campo **Note** per annotazioni visibili solo nel pannello (non sul sito pubblico).

## Esportare in CSV

Dalla lista **Leads** apri il menu azioni e scegli **Export**.

1. Seleziona il formato **CSV**
2. Scegli i campi da includere (o lascia il set predefinito)
3. Scarica il file

Puoi esportare tutta la lista o solo le righe selezionate.
