La sezione **In evidenza** della homepage mostra contenuti selezionati manualmente tramite un flag dedicato.

## Cosa può andare in evidenza

| Tipo contenuto | Dove si imposta |
| --- | --- |
| Lauree | Scheda laurea → campo **In evidenza** |
| Master | Scheda master → campo **In evidenza** |
| Corsi online | Scheda corso → campo **In evidenza** |
| Aree tematiche | Scheda area → campo **In evidenza** |
| Articoli blog | Scheda articolo → campo **In evidenza** |
| Testimonianze | Scheda recensione → **In evidenza** + **Ordine** |

## Passi operativi

<Steps>
  <Step title="Apri la scheda">
    Vai alla laurea, master, corso o area che vuoi promuovere.
  </Step>
  <Step title="Attiva In evidenza">
    Spunta il campo **In evidenza** (di solito nella sidebar o nei dati principali).
  </Step>
  <Step title="Salva">
    Attendi l'aggiornamento del sito (rebuild automatico).
  </Step>
  <Step title="Verifica homepage">
    Controlla [laureasi.it](https://laureasi.it) che il contenuto compaia nella sezione corretta.
  </Step>
</Steps>

## Quanti contenuti mostrare?

Il numero di slot in homepage dipende dal layout del frontend. Se troppi contenuti sono in evidenza, il sito ne mostrerà un sottoinsieme (es. i più recenti o per tipo).

<Note>
Non tutti i tipi di contenuto compaiono nella stessa sezione della homepage. Lauree, master e corsi hanno aree dedicate; le testimonianze usano anche il campo **Ordine**.
</Note>
