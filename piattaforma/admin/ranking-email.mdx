# Funzione per inviare risultati di orientamento via email

## Uso

```typescript
import { sendRankingResultsEmail } from "@/services/email";

await sendRankingResultsEmail({
  to: "user@example.com",
  results: [
    {
      title: "Laurea in Informatica",
      courseType: "Laurea Magistrale",
      entityLabel: "Laurea",
      university: "Università di Milano",
      reason: "Perfetto match con il tuo profilo e i tuoi obiettivi",
      url: "https://laureasi.it/universita/milano/informatica",
      priceLabel: "€ 2.000-3.000/anno",
      rank: 1,
    },
    // ... altri risultati
  ],
  quiz: {
    obiettivo: "Specializzarmi in Intelligenza Artificiale",
    areaInteresse: "Informatica",
    livello: "Laurea Magistrale",
    situazione: "Lavoro a tempo pieno",
    tempo: "Part-time",
  },
});
```

## Struttura dei dati

### `RankingResult`

```typescript
interface RankingResult {
  title: string; // Nome della laurea/master
  courseType?: string; // Es: "Laurea Magistrale", "Master"
  entityLabel?: string; // "Laurea" | "Master" | "Corso"
  university: string; // Nome ateneo
  reason: string; // Motivo della raccomandazione
  url: string; // Link al corso
  priceLabel?: string; // Es: "€ 1.000-2.000/anno"
  appliedBenefitLabel?: string | null;
  rank?: number;
}
```

### `OrientationQuiz`

```typescript
interface OrientationQuiz {
  obiettivo: string; // Obiettivo di studi
  areaInteresse: string; // Area di interesse
  livello: string; // Livello di studio
  situazione?: string; // Situazione (es: "Studente", "Lavoratore")
  tempo?: string; // Tempo disponibile (es: "Part-time", "Full-time")
}
```

## Integrazione con l'endpoint `/api/trova-laurea/recommend`

Dopo aver ottenuto i risultati, puoi inviare l'email così:

```typescript
// In route.ts di /api/trova-laurea/recommend
import { sendRankingResultsEmail } from "@/services/email";

export async function POST(request: Request) {
  // ... logica di ranking ...

  const results = topCandidates.map((c) => ({
    title: c.title,
    courseType: c.courseType,
    entityLabel: c.entityLabel,
    university: c.university,
    reason: c.reason,
    url: c.url,
    priceLabel: c.priceLabel,
    rank: c.rank,
  }));

  // Invia email se l'utente ha fornito email
  if (body.userEmail) {
    await sendRankingResultsEmail({
      to: body.userEmail,
      results,
      quiz: body,
    });
  }

  return NextResponse.json({ success: true, data: { results } });
}
```

## Email in HTML

L'email include:

- Header con logo LaureaSì (PNG white su fondo primario `#A6192E`)
- Riepilogo profilo dell'utente (obiettivo, area, livello, ecc.)
- CTA primaria **Apri i tuoi risultati** (se `resultUrl` è presente)
- Card risultati allineate al layout sito: badge entity, Scelta #N, titolo, ateneo, motivo, prezzo, CTA outline **Vedi dettagli**
- Badge **Prima scelta** sulla prima card (senza punteggio %)
- Sezione **Altri percorsi compatibili** oltre i top 3
- CTA secondaria **Prenota una consulenza** → `/prenota-consulenza`
- Design responsive, palette ufficiale Pantone 187 C / 440 C + accent teal

## Gestione errori

Se l'invio fallisce, la funzione:

- Lancia un errore (catch con try/catch)
- Loga l'errore con prefisso `[Ranking Email]`
- Non blocca il flusso dell'API (chiama la funzione in background)

```typescript
// Esempio: invia ma non bloccare se fallisce
sendRankingResultsEmail({ ... }).catch(error => {
  console.error('Ranking email failed:', error)
  // Opzionale: notifica admin
})
```
